# PythonMagic

**Where Code Becomes Magic** — Interactive Python-powered playground running entirely in your browser.

## Features

- 🎮 **Games** — Snake, Pong, Tower Defense & more
- 🧩 **Widgets** — Live clocks, calculators, data visualizations
- 🎆 **Magic Visuals** — Particle storms, waveform generators, neon spirals
- 🔮 **Kaleidoscope** — Full-screen immersive kaleidoscope visualizer
- 🎧 **Audio Reactive** — Visualizations driven by microphone or music
- 🌊 **Fluid Dynamics** — Interactive fluid simulation
- 🎨 **Generative Art** — Algorithmic art engine

## Tech Stack

- **Vanilla JavaScript** — No frameworks, no build step
- **HTML5 Canvas & WebGL** — Hardware-accelerated graphics
- **Web Audio API** — Real-time audio analysis
- **CSS3** — Custom properties, animations, glassmorphism

## Usage

Just open `index.html` in a browser. No server required.
